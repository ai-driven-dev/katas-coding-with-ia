# Projet TypeScript

## Prérequis

- Node.js (version 14.x ou supérieure)
- npm (version 6.x ou supérieure)

## Installation

1. Installez les dépendances :

    ```bash
    npm install
    ```

2. lancez les tests:

    ``` bash
    npm run test -- --watch
    
    ```
